# Healthcare SQL Analysis

## Project Overview

This project analyzes healthcare operational data using SQL to identify patient trends, department performance, and operational insights.

## Skills Demonstrated

- Joins
- CTEs
- Aggregate Functions
- Window Functions
- Data Analysis

## Business Questions

1. Which department has the highest patient volume?
2. What are the admission trends?
3. Which departments have the longest average stay?
4. How can operational efficiency be improved?

## Tools Used

- SQL
- Excel
