# E-Commerce SQL Analysis

## Project Overview

This project analyzes e-commerce sales data using SQL to identify revenue trends, customer behavior, and product performance.

## Skills Demonstrated

- Joins
- CTEs
- Aggregate Functions
- Window Functions
- Data Analysis

## Business Questions

1. Which products generate the most revenue?
2. Which customer segments are most profitable?
3. What are the monthly sales trends?
4. Which regions perform best?

## Tools Used

- SQL
- Excel
