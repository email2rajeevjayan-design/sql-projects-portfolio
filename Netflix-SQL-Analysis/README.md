# Netflix SQL Analysis

## Project Overview

This project analyzes Netflix content data using SQL to identify trends in genres, ratings, countries, and content growth.

## Skills Demonstrated

- Joins
- CTEs
- Aggregate Functions
- Window Functions
- Data Analysis

## Business Questions

1. Which genres are most common?
2. Which countries produce the most content?
3. How has Netflix content grown over time?
4. What are the most common ratings?

## Tools Used

- SQL
- Excel
